Prototype Anglo Concertina Fingering Generator

Caveats:

  1. The button selection algorithm is currently very poor. Lots of hops. To be fixed soon!
  2. Only Jeffries button layouts supported right now.
  3. ABC with embedded quotes not supported yet.
  4. Not all key signatures are supported yet.
  5. Accidentals are not carried to other unmarked notes in the same measure.

Instructions

Paste an ABC style tune into the box below, then hit "Generate Fingerings". Review the generated fingerings and manually correct any errors in the result box.

Then copy the resulting annotated ABC into your clipboard and use it on many of the fine PDF generator sites for ABC notation. My personal favorite is Draw The Dots.

For button notation, see the Button Notation Chart
